# NodeJS Express TypeScript Session REST API

A REST API using NodeJS with Express, TypeScript, and sessions.

From: https://www.youtube.com/watch?v=b8ZUb_Okxro

# Installation

Create an account at: https://cloud.mongodb.com/

copy config.bak.json to config.json and enter your database
admin credentials. You can also change the secret for the salt,
and the cookie's name.

Run:

```
npm install
```

# Start the Server

```
npm start
```
